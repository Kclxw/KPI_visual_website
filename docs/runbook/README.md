# 运维开发手册

本目录包含系统的运维与开发操作手册。

## 文档列表

- **local-dev.md** - 本地开发环境搭建指南
- **startup.md** - 项目启动指南（本地）
- **deploy.md** - 生产环境部署手册
- **troubleshooting.md** - 常见问题排查指南

## 快速开始

### 新人开发环境搭建
1. 阅读 `local-dev.md`
2. 安装依赖环境
3. 执行一键启动脚本
4. 验证各模块运行状态

### 生产部署
1. 阅读 `deploy.md`
2. 准备部署清单
3. 按流程执行部署
4. 验证服务状态

### 问题排查
1. 先查看 `troubleshooting.md` 常见问题
2. 检查日志文件
3. 联系相关负责人

## 注意事项

- 所有生产操作必须经过审批
- 部署前必须在测试环境验证
- 重要操作需要备份
- 记录所有操作日志

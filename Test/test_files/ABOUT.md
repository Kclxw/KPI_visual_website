# 目录职责
本目录用于：存放额外的上传测试文件，供手工联调时快速复用，不参与脚本自动生成逻辑。

# 关键模块 / 文件一览
- `test_ifir_detail.xlsx`：可直接用于上传接口测试的 IFIR 明细文件样本。

# 主要数据流 / 调用链
- 人工联调时可从本目录挑选样本文件，通过上传接口或前端上传页面验证文件导入流程。

# 与上层 / 下层目录的关系
- 上层：`Test/` 目录中的测试脚本和人工测试流程会引用这里的文件。
- 下层：无进一步子目录。

# 运行方式 / 配置项
- 本目录文件没有被统一脚本自动扫描；若要使用，需要在测试脚本中手动指定路径或手工上传。

- 若样本文件字段结构过旧，上传接口可能在 ETL 阶段失败。
